# Oracle EPM Planning - Groovy and Calc Script Examples

This repository contains practical examples of Oracle EPM Planning / EPBCS automation:
- Groovy business rules (EPM Groovy / EpmScript style)
- Essbase calc scripts (Planning business rules)
- Patterns for validations, Smart List-driven logic, spreading, and workflow-style calculations

## What this demonstrates (ACE product usage)
- Oracle EPM Planning Groovy business rule development
- Reading Smart List values and driving form behavior
- Writing back values to the cube using supported approaches
- Essbase calc script patterns with FIX scoping and performance considerations

## Contents
- `groovy/` - Groovy business rule examples
- `essbase/` - Essbase calc scripts
- `docs/` - Explanations and setup notes
- `samples/` - Sample metadata and assumptions

## Notes
- All examples use sanitized/generic metadata and do not include any client-specific identifiers.
- Replace dimension/member names to match your environment before using.
